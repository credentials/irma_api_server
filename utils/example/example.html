IRMA demo disclosure

This page demonstrates IRMA attributes disclosure, using the IRMA API server that you specify below. Be sure to enter an IP address or domain name that your IRMA app is able to connect to (i.e., localhost will generally not work).

Examples (assuming your IP address is 1.2.3.4):

  • If you are running the API server using docker: 1.2.3.4:8088
  • If you are running the API server directly or using ansible: 1.2.3.4:8088/irma_api_server
